Project Summary: Mobile Sales Analysis & Forecasting

1. Objective: 

The goal of this project was to analyze 1,000 transaction records to identify revenue drivers and proactively plan for future sales trends.

2. Key Insights:

Total Revenue: The analysis revealed a total revenue of $40.22 million.

Top Product: The "huge" model was the primary source of income, generating $282,863.97.

Best Location: Lake Amanda was the top-performing region with $185,827.20 in sales.

3. Data Science Workflow:

Data Preparation: Converted raw string dates to datetime and performed feature engineering to extract Year and Month.

Machine Learning: Built a Linear Regression model using scikit-learn.

Forecasting: Used a numerical TimeIndex to predict sales for the next quarter.

4. Business Recommendation :

The model predicts a slight downward trend for the upcoming quarter. I recommend a targeted marketing campaign in Lake Amanda to counter this dip.
